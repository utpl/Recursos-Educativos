🏭
Duopolio: Escenarios
Aplicación matemática con costos marginales constantes (146) y demanda P=340−Q. Se muestran los resultados de cada modelo de oligopolio: Cournot, Stackelberg, Cártel y Bertrand.
📌 Modelos analizados
  • Cournot – competencia simultánea en cantidades
  • Stackelberg – líder mueve primero, seguidor reacciona
  • Cártel – colusión, maximizan beneficio conjunto
  • Bertrand – competencia en precios → P=CMg
📊
Cournot
Cantidades simultáneas
Función de demanda inversa: P = 340 − (q₁+q₂). Beneficios: π₁ = (340−Q)q₁ − 146q₁ = 194q₁ − q₁² − q₁q₂.
∂π₁/∂q₁ = 194 − 2q₁ − q₂ = 0 → q₁ = 97 − q₂/2 (función reacción)
Simétricamente: q₂ = 97 − q₁/2 → Resolviendo: q₁ = q₂ = 194/3 ≈ 64.67
Resultados:
Q = 129.34 | P = 340 − 129.34 = $210.66
π₁ = π₂ = (210.66−146)×64.67 ≈ $4181.57
🚀
Stackelberg (líder empresa 1)
Ventaja de primer movimiento
La seguidora (empresa 2) actúa según Cournot: q₂ = 97 − q₁/2. La líder incorpora esa reacción.
π₁ = [340 − (q₁ + 97 − q₁/2)]q₁ − 146q₁ = (194q₁ − q₁²)/2
CPO: 97 − q₁ = 0 → q₁ = 97 → q₂ = 97 − 97/2 = 48.5
Q = 145.5 | P = 340−145.5 = $194.5
π₁ = (194.5−146)×97 = $4704.5 | π₂ = (194.5−146)×48.5 = $2352.25
🤝
Cártel (colusión perfecta)
Maximización conjunta
Actúan como monopolio: IT = (340−Q)Q → IMg = 340 − 2Q. Igual a CMg=146.
340 − 2Q = 146 → Q = 97 → P = 340 − 97 = $243
Beneficio total: (243−146)×97 = $9409. Si se reparten equitativamente: π₁=π₂=$4704.5
🔄
Cártel (planteamiento alternativo)
Beneficio conjunto
Maximizar ∑π = (340−Q)Q − 146Q, con Q = q₁+q₂. Derivando respecto a Q: 340 − 2Q − 146 = 0 → Q = 97. Mismo resultado: precio $243, beneficio total $9409. Se observa que colusión maximiza beneficios conjuntos.
⚡
Bertrand (competencia en precios)
Resultado competitivo
Si las empresas compiten en precios con producto homogéneo y costos idénticos, el equilibrio es P = CMg = 146.
340 − Q = 146 → Q = 194. Cada empresa produce 97 unidades, precio $146 → π₁ = π₂ = 0.
Beneficio económico nulo, producción máxima (194) y precio igual al costo marginal.
📈
Comparativa de resultados
Resumen ejecutivo
Cournot: q=64.67 c/u, P=$210.66, beneficio unitario $4181.57
Stackelberg: Líder q=97, seguidor 48.5, P=$194.5, π₁=$4704.5, π₂=$2352.25
Cártel: Q=97, P=$243, beneficio conjunto $9409 (máximo)
Bertrand: P=$146, Q=194, beneficio nulo.
La colusión genera mayor precio y beneficio, mientras Bertrand iguala a competencia perfecta. Stackelberg otorga ventaja al líder.
💡 Conclusión: El modelo de Cournot produce beneficios intermedios, Stackelberg beneficia al líder, el cártel maximiza utilidades conjuntas y Bertrand genera eficiencia de mercado con beneficio cero.
