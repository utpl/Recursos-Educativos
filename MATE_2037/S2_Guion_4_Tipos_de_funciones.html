Tipos de Funciones A continuación se presentan definiciones y formas representativas de distintos tipos de funciones para reconocer sus características esenciales. Seleccione cada función con los botones de colores para explorar su definición, fórmula y representación gráfica.
CONSTANTE
f(x) = c
Función que siempre da el mismo valor de salida y independientemente del valor de entrada x. Su gráfica es una línea horizontal paralela al eje x.
Dominio: todos los reales ℝ
Rango: { c } — un único valor
Pendiente = 0 en todo su dominio
Ejemplo: f(x) = 3 siempre retorna 3
GRÁFICA · f(x) = 3
LINEAL
f(x) = mx + b
Relación de cambio constante entre x e y. Su gráfica es una línea recta. El coeficiente m es la pendiente y b es el intercepto en y.
Dominio: ℝ — Rango: ℝ
m > 0 → creciente; m < 0 → decreciente
m = 0 → función constante
Ejemplo: f(x) = 2x + 1
GRÁFICA · f(x) = 2x + 1
CUADRÁTICA
f(x) = ax² + bx + c
Función polinómica de segundo grado. Su gráfica es siempre una parábola: abre hacia arriba si a > 0, hacia abajo si a < 0.
Dominio: ℝ — Rango depende de a
Vértice: punto mínimo o máximo
Eje de simetría: x = −b/2a
Ejemplo: f(x) = x² − 2x − 3
GRÁFICA · f(x) = x²
RACIONAL
f(x) = p(x)/q(x)
Función expresada como cociente de dos funciones polinómicas. Presenta asíntotas donde el denominador se anula.
Dominio: ℝ − { x: q(x)=0 }
Asíntota vertical donde q(x) = 0
Asíntota horizontal si grado(p) ≤ grado(q)
Ejemplo: f(x) = 1/x (hipérbola)
GRÁFICA · f(x) = 1/x
EXPONENCIAL
f(x) = aˣ
La variable independiente x está en el exponente. La base a es una constante real positiva diferente de 1 (a > 0, a ≠ 1).
Dominio: ℝ — Rango: (0, +∞)
a > 1 → creciente; 0 < a < 1 → decreciente
Asíntota horizontal y = 0
Ejemplo: f(x) = 2ˣ (crecimiento)
GRÁFICA · f(x) = 2ˣ
LOGARÍTMICA
f(x) = log_a(x)
Función inversa de la exponencial. La base a es un número real positivo diferente de 1 (a > 0, a ≠ 1). Crece sin límite pero muy lentamente.
Dominio: (0, +∞) — Rango: ℝ
Asíntota vertical: x = 0
Pasa por el punto (1, 0) siempre
Ejemplo: f(x) = log₂(x)
GRÁFICA · f(x) = log₂(x)
