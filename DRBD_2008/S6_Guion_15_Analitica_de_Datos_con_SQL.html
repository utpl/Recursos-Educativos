functions Funciones de Agregación

  • • Trabajan sobre un conjunto de valores y devuelven un solo valor.
  • • Ignoran los valores NULL (excepto COUNT(*)).
  • • Se usan frecuentemente con la cláusula GROUP BY.

Layers Operaciones de Conjuntos

  • • Combinan los resultados de dos o más consultas SELECT.
  • • Las consultas deben tener el mismo número y tipo de columnas.
  • • Ayudan a comparar listas de datos de diferentes tablas.
