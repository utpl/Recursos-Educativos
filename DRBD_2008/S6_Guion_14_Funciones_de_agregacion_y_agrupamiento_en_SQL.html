FUNCIONES DE AGREGACIÓN Y AGRUPAMIENTO EN SQL

Estas funciones permiten resumir grandes volúmenes de datos mediante cálculos específicos, mientras que el agrupamiento organiza la información para su análisis; a continuación, se presentan sus componentes clave y su aplicación en consultas SQL.

format_list_numbered COUNT()

  • Cuenta filas o valores no nulos.
  • Con DISTINCT cuenta valores únicos.
SELECT COUNT(columna) FROM tabla;

calculate SUM()

  • Calcula la suma de una columna numérica.
  • Ignora valores NULL.
SELECT SUM(ventas) FROM pedidos;

analytics AVG()

  • Calcula el promedio de valores numéricos.
  • Se combina con GROUP BY para análisis por grupos.
SELECT AVG(precio), categoria FROM productos GROUP BY categoria;

swap_vert MIN() / MAX()

  • Identifican el valor mínimo y máximo.
  • Aplicables a datos numéricos, fechas o texto.
SELECT MIN(fecha), MAX(fecha) FROM eventos;

group_work GROUP BY (Agrupamiento)

  • Agrupa filas con valores comunes.
  • Permite aplicar funciones de agregación por grupo.
  • Las columnas sin agregación deben incluirse en GROUP BY.
SELECT categoria, COUNT(*) FROM productos GROUP BY categoria;

filter_alt WHERE vs HAVING

filter_list WHERE
  • Filtra filas antes del agrupamiento.
WHERE precio > 100
group HAVING
  • Filtra grupos después del agrupamiento.
  • Permite condiciones sobre funciones de agregación.
HAVING COUNT(*) > 5
