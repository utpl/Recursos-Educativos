Fundamentos de Base de Datos • Caso Práctico

📋 Caso de Estudio

Una institución educativa gestiona información académica con estructuras variables, consultas frecuentes y relaciones entre entidades. El equipo evalúa migrar de un modelo relacional a NoSQL para mejorar rendimiento, escalabilidad y flexibilidad. Se consideran alternativas como documentos, clave-valor, columnas y grafos, además del uso de MongoDB para operaciones básicas de inserción, consulta, actualización y eliminación. También se analizan ventajas como alta disponibilidad y desventajas como consistencia eventual y limitaciones en JOIN. El desafío consiste en seleccionar modelos adecuados según necesidades específicas de datos y consultas. Se requiere tomar decisiones fundamentadas para optimizar el sistema.
