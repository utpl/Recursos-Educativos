🎯 INICIO RÁPIDO
================================================================================

1. ABRIR EL CATÁLOGO DE RECURSOS
   • Abre index.html en tu navegador
   • Verás todos los 325 recursos organizados por carpeta
   • Usa la barra de búsqueda para filtrar

2. VER ESTADÍSTICAS
   • En index.html, haz clic en "📊 Ver Reporte"
   • Se abre reporte.html con gráficos y análisis
   • Incluye distribución por carpetas

3. DESCARGAR DATOS EN EXCEL
   • En index.html, haz clic en "📥 Descargar Excel"
   • Se descarga recursos_data.xlsx
   • Contiene 3 hojas: Resumen, Distribución, Recursos

🔄 ACTUALIZAR RECURSOS
================================================================================

OPCIÓN 1: Script Automático (Recomendado)
  • Ejecuta: python actualizar_todos.py
  • Actualiza JSON y Excel automáticamente
  • Genera reporte completo

OPCIÓN 2: Manual
  • Ejecuta: python generar_recursos.py
  • Luego: python generar_excel.py
  • Los HTML se actualizan automáticamente

OPCIÓN 3: Interfaz Web
  • index.html y reporte.html cargan datos dinámicamente
  • Solo necesitas actualizar recursos_data.json
  • No es necesario regenerar HTML

❓ PREGUNTAS FRECUENTES
================================================================================

P: ¿Cómo agrego nuevos recursos?
R: Coloca el archivo .html en la carpeta correspondiente
   Luego ejecuta: python actualizar_todos.py

P: ¿El index.html se actualiza automáticamente?
R: Sí, carga datos desde recursos_data.json dinámicamente

P: ¿Puedo integrar esto en Canvas?
R: Sí, usa index.html como iFrame (configurar ancho 800px)

P: ¿Qué navegadores son compatibles?
R: Todos los navegadores modernos (Chrome, Firefox, Safari, Edge)

P: ¿Se puede modificar el diseño?
R: Sí, edita los estilos CSS en index.html o reporte.html

P: ¿Dónde están alojados los recursos?
R: En las carpetas de contenido (ADMI_1134/, DRBD_2008/, etc.)

P: ¿Es necesario tener Python instalado?
R: Solo si quieres actualizar recursos automáticamente
   Para ver los recursos: solo necesitas un navegador web

🚀 INTEGRACIÓN CON CANVAS
================================================================================

1. En tu curso de Canvas, ve a: Configuración → Pestañas personalizadas
2. Crea una nueva pestaña personalizada
3. En la URL, apunta a tu servidor donde está alojado index.html
4. Configura:
   - Ancho: 800px
   - Alto: 1200px (aproximadamente)

5. El sistema se verá perfecto en Canvas:
   - Buscador funciona
   - Todos los enlaces funcionan
   - Diseño responsive
   - Cargas rápidas

📞 SOPORTE
================================================================================

Si encuentras problemas:

1. Verifica que recursos_data.json exista
2. Abre la consola del navegador (F12)
3. Busca mensajes de error
4. Ejecuta nuevamente los scripts de generación
5. Limpia la caché del navegador (Ctrl+Shift+Delete)

Errores comunes:
  • "No se cargan recursos" → Comprueba recursos_data.json
  • "Los estilos se ven raros" → Limpia caché
  • "404 en archivos" → Verifica las rutas relativas
