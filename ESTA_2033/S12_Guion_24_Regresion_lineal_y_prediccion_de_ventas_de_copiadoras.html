📠 CASO: PREDICCIÓN DE VENTAS DE COPIADORAS MEDIANTE REGRESIÓN LINEAL
Una empresa dedicada a la venta de copiadoras desea analizar si el número de llamadas telefónicas realizadas por sus vendedores permite predecir la cantidad de copiadoras vendidas. Después de recopilar la información, se construyó un modelo de regresión lineal simple con la siguiente ecuación:
Y = 19.2821 + 1.0079X
Y Cantidad de copiadoras vendidas
X Número de llamadas telefónicas realizadas
19.28 Intercepto (valor cuando X = 0)
1.01 Pendiente (incremento por cada llamada)
Con este modelo, la empresa desea estimar cuántas copiadoras podría vender según el número de llamadas realizadas por sus vendedores.
🏆
¡Felicidades!
Ha concluido con éxito el análisis del caso.
✓ La regresión lineal permite construir ecuaciones para describir relaciones entre variables y realizar predicciones.
✓ El intercepto indica el valor esperado de Y cuando X es igual a cero, mientras que la pendiente muestra cuánto cambia Y por cada unidad adicional de X.
Este tipo de análisis resulta útil para apoyar la toma de decisiones y estimar resultados futuros a partir de datos observados.
